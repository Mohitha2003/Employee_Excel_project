# 📊 Employee Analysis Dashboard (Excel Project)

## 🔍 Project Overview

This project focuses on cleaning, transforming, and analyzing an employee dataset using Excel. The objective was to generate meaningful insights and present them through an interactive dashboard.

---

## 🧹 Data Cleaning & Preparation

* Standardized data types (text, numeric, and date formats)
* Applied **Data Validation** to ensure data accuracy
* Handled missing values:

  * Replaced missing **Salary** and **Age** values using the **average**
  * Filled missing **Gender** values as *"Not Specified"*
* Detected outliers using the **IQR (Interquartile Range) method**

  * No significant outliers were found in the dataset
* Used **Text to Columns** to properly format the Joining Date column
* Removed inconsistencies and ensured data quality

---

## 📊 Data Analysis

Performed analysis using Pivot Tables:

* Employee count by city
* Average salary by department
* Gender distribution
* Top 5 highest-paid employees

---

## 📈 Dashboard Features

* KPI Cards:

  * Total Employees
  * Total Salary
* Gender Distribution (Donut Chart)
* Average Salary by Department (Bar Chart)
* Top 5 Highest Salaries
* Employees by City
* Interactive filters using **Slicers and Timeline**

---

## 🎯 Key Insights

* Identified departments with highest average salaries
* Analyzed employee distribution across cities
* Observed balanced gender distribution with minimal missing values
* Verified data consistency with no major outliers

---

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Conditional Formatting
* Data Validation
* IQR Method for Outlier Detection
---

## 🚀 Conclusion

This project demonstrates end-to-end data analysis in Excel — from data cleaning to dashboard creation — ensuring data quality and delivering actionable insights.
